# Exames anteriores

Os PDFs originais de onde saíram as perguntas do banco (drill/exame) e os
enunciados do Grupo II.

- **SCOMP2425_Modelo_A.pdf** — o exame modelo indicado pelos docentes. É A
  referência da estrutura: 10 MCQ + 2 programas.
- **SCOMP-Reference-Sheet.pdf** — a folha de referência da API que é fornecida
  NO exame. Treina com ela ao lado: no dia, não precisas de decorar assinaturas.
- **Ep{Normal,Recurso,Especial}_24_25.pdf** e **23_24** — épocas anteriores,
  mesma estrutura do modelo.
- Os `*_22_23_oldVersion.pdf` têm estrutura antiga — só as folhas de resposta,
  sem o caderno de perguntas; serviram pouco para o banco.

Que padrão de Grupo II caiu em cada época: ver `../patterns/EXAM-HISTORY.md`.
